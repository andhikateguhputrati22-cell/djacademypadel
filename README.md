# DJ Academy Padel Booking - Mobile Clean

Versi ini dibuat untuk update repository GitHub/Netlify agar tampilan lebih simpel, tetap memakai logo DJ Academy, dan fokus mobile.

## Urutan tampilan
1. Logo DJ Academy
2. Slogan
3. Harga dan durasi
4. Paket coaching
5. Form booking
6. Kirim ke WhatsApp

## Cara upload ke GitHub
1. Extract ZIP.
2. Upload semua isi folder ke repository.
3. Pastikan index.html berada di root repository, bukan di dalam folder bersarang.
4. Commit changes.
5. Netlify akan auto deploy jika repository sudah terhubung.
